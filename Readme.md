# 🚀 Space Themed Feedback

This is a small project I built to make collecting feedback a little more fun instead of using a plain form. The idea was to create a feedback system with a space-themed interface so users can leave feedback in a more engaging environment.

The project stores submitted feedback and keeps it available for later review and analysis.

## What it does

* Collects user feedback
* Uses a space-themed design and visuals
* Saves responses into CSV and Excel files
* Keeps feedback organized and easy to access
* Simple project structure for learning and customization

## Run locally

Clone the repository:

```bash
git clone https://github.com/maheshairborne/space_themed_feedback.git
```

Move into the project folder:

```bash
cd space_themed_feedback
```

Run the application:

```bash
python main.py
```

## Why I made this

This project was created for an event that had a space-themed concept. Instead of building a regular feedback form, I wanted to match the theme and make the experience more engaging and interactive. The goal was to create something that fit the event atmosphere while still collecting and storing feedback in a simple and organized way.
